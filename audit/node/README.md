## 如何安装与使用

### vueapp_pc

```sh
# 下载 vueapp_pc
git clone https://gitee.com/cqzqxq_sjl/vueapp_pc

# 进入 vueapp_pc 目录
cd vueapp_pc

# 安装依赖 推荐使用 yarn
npm install

# 本地运行 vueapp_pc
npm run serve

# 项目打包（）
npm run build
```
